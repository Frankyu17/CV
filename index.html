HexOust Game

Game Design

×

HexOust is a two-player strategic board game developed in Java. Players (Yellow vs. Red) take turns placing tokens on a hexagonal grid. The core mechanic requires players to capture smaller adjacent opponent groups with their own larger group of connected tokens. The game enforces strict rules, provides visual feedback, and includes features like an undo function and win tracking.

Language

Java

GUI Framework

JavaFX

This was a group project developed over five Agile sprints. We used Gantt charts for sprint planning, tracking objectives like core logic and final polishing. I was primarily responsible for the core game logic and user interface interactivity.

My main contributions included:

  • Designing and implementing the complex capture logic and game rules that form the core gameplay mechanics
  • Programming the algorithm to find connected token groups on the hexagonal grid, enabling strategic gameplay
  • Developing the comprehensive validation system that accepts or rejects moves based on game rules
  • Enhancing UI/UX by implementing visual feedback systems, including hexagon highlighting on hover
